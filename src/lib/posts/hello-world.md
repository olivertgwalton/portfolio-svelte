## Introduction

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents. Created by John Gruber in 2004, Markdown is now one of the world’s most popular markup languages.

### Why Markdown?

- **It's portable.** You can open Markdown-formatted files in any application.
- **It's platform independent.** You can create Markdown-formatted text on any device running any operating system.
- **It's future proof.** Even if the application you’re using stops working, you’ll still be able to read your Markdown-formatted text using a text editor.
